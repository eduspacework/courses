GAI

Generative AI Training Program — Certification Course

Duration: 20 hours (5 × 4-hour sessions) • Level: Beginner • Delivery: Instructor-led / Online live

Course Overview

A practical, project-focused introduction to Generative AI covering core concepts, prompt engineering (CRAFT), text & image generation, basic chatbot design, and Python API integration.

Learning Objectives

  • Explain foundational AI concepts and distinctions within Generative AI.
  • Identify AI types by capability and functionality with real-world examples.
  • Apply the CRAFT prompt engineering method for effective outputs.
  • Use AI APIs from Python to generate text, code, and images.
  • Build and demo a simple AI-powered application.

Who Should Attend

  • Students and professionals with basic programming knowledge.
  • Product managers, content creators, and developers seeking practical Generative AI skills.

Course Outline (5 Sessions)

Session 1 — Introduction to AI & Generative AI (4 hrs)

  • AI fundamentals, history, and definitions
  • Types by capability: Narrow AI, AGI, Super AI
  • Functional classes: Reactive, Limited Memory, Theory of Mind
  • Generative AI modalities: text, code, images — demo & quiz

Session 2 — Prompt Engineering with CRAFT (4 hrs)

  • CRAFT: Context, Role, Action, Format, Tone/Tweak
  • Techniques: chain-of-thought, few-shot, temperature, tokens
  • Labs: prompts for summarization, emails, coding

Session 3 — AI APIs & Python Integration (4 hrs)

  • API basics: auth, request/response, rate limits, error handling
  • Hands-on: call text and image generation APIs from Python
  • Mini lab: CLI tool that produces formatted outputs

Session 4 — Text & Image Generation, Evaluation (4 hrs)

  • Model capabilities, evaluation metrics, safety basics
  • Labs: generate and iterate on blog intros, code, and images
  • Best practices for reproducibility and cost control

Session 5 — Chatbot Basics & Final Projects (4 hrs)

  • Conversational design: state, context, fallback strategies
  • Integration demo (webhook/REST) and simple UI example
  • Final project presentations and certification
