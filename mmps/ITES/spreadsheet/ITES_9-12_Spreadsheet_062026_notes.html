Lesson 1: Introduction to LibreOffice Calc

Core Concepts (Study Notes)

  • Spreadsheet Basics: A digital ledger used to store, organize, and manipulate data.

  • Grid Structure: Data is entered into Cells, which are the intersections of horizontal Rows (numbered 1, 2, 3...) and vertical Columns (lettered A, B, C...).

  • Workbook vs. Worksheet: A Workbook is the entire file (like a binder), while a Worksheet is an individual page within that file.

  • File Management: Always save files with clear, descriptive naming conventions (e.g., Grade12_Attendance_YourName.ods) for easy retrieval.

Lesson 2: Formatting Worksheets

Core Concepts (Study Notes)

  • Visual Organization: Unformatted data is difficult to read. Formatting improves readability and professional presentation.

  • Cell Formatting: Includes changing font styles, sizes, and colors to emphasize important data (like headers).

  • Borders and Alignment: Used to separate data sets visually. Text can be aligned left, center, or right within a cell.

  • Row/Column Adjustment: Modifying the height and width ensures text does not get cut off.

  • Merge & Center: A tool used to combine multiple cells into one large cell, perfect for main titles.

Lesson 3: Basic Formulas and Calculations

Core Concepts (Study Notes)

  • Formula Structure: Every formula in Calc must begin with an equals sign (=).

  • Basic Operators: Addition (+), Subtraction (-), Multiplication (*), and Division (/). Example: =A1+B1.

  • Cell Referencing: Instead of typing numbers (e.g., =10+5), use the cell addresses (e.g., =C2+D2). This ensures the answer updates automatically if the original data changes.

  • AutoFill: A massive time-saver. By dragging the small square at the bottom right of a selected cell, you can copy a formula down an entire column.

Lesson 4: Functions in Calc

Core Concepts (Study Notes)

  • Functions vs. Formulas: Functions are pre-built formulas provided by Calc to perform specific, often complex, calculations instantly.

  • Key Functions to Know:

    • =SUM(range): Adds all numbers in a range.

    • =AVERAGE(range): Calculates the mean of a range.

    • =MAX(range): Finds the highest value.

    • =MIN(range): Finds the lowest value.

    • =COUNT(range): Counts how many cells contain numbers.

Lesson 5: Charts and Graphs

Core Concepts (Study Notes)

  • Data Visualization: Humans process visual information faster than text. Charts turn rows of numbers into actionable insights.

  • Bar/Column Charts: Best for comparing different categories (e.g., comparing marks between students).

  • Pie Charts: Best for showing parts of a whole (e.g., percentage of budget spent on different categories).

  • Line Charts: Best for showing trends over time (e.g., sales growth over a month).

  • Formatting: A chart must always have a clear Title, Axis Labels, and a Legend to be easily interpreted.

Lesson 6: Sorting and Filtering Data

Core Concepts (Study Notes)

  • Sorting Data: Rearranging rows based on the contents of a specific column. You can sort alphabetically (A-Z or Z-A) or numerically (Highest to Lowest).

  • Filtering Information: Temporarily hiding rows that do not meet specific criteria. This allows you to focus only on the data you need at that moment.

  • Custom Sorting: Sorting by multiple levels (e.g., Sort by Grade level first, and then Alphabetically by Last Name within that grade).

Lesson 7: Data Validation and Conditional Formatting

Core Concepts (Study Notes)

  • Data Validation: A rule set up to restrict what users can type into a cell. This prevents errors (e.g., creating a drop-down menu so a user can only select "Pass" or "Fail").

  • Conditional Formatting: Making a cell change its appearance (color, font) automatically based on the data inside it.

  • Use Cases: Instantly turning cells red if a budget is exceeded, or turning them green if a sales target is met.
